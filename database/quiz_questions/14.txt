TITLE:
Для чего используется ключевое слово finally в С++?
FIRST:
Для декларации членов класса, которые не могут быть изменены после вызова конструктора класса
SECOND:
Такого ключевого слова нет в С++
THIRD:
Для объявления блока кода, идущего после блока try и, опционально, после блоков catch, содержащий код, который должен выполниться в любом случае
FORTH:
Для запрета переопределения виртуальной функции или наследования определенного класса
CORRECT:
SECOND
