TITLE:
Скомпилируется ли следующий код?
---
template <class T=void>
struct A {
  int a;
};

template <>
struct A<int> {
  char c; // 2
};

void Test() {
  A<>().a; // 3
  A<int>().c; // 4
}
---
FIRST:
Да
SECOND:
Нет, т.к. ошибка в строке //3
THIRD:
Нет, т.к. ошибка в строке //4
FORTH:
Нет, т.к. специализированный шаблон класса содержит члены, отличающиеся от определенного выше шаблона класса (//2)
CORRECT:
FIRST
